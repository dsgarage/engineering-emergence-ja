## 翻訳範囲

- 第1章: イントロダクション（ゲームの定義、メカニクス、創発とプログレッション）
- 第2章: ルール、表現、リアリズム（イコニック・インデクシカル・シンボリックなシミュレーション）
- 第3章: ゲームデザイン理論（MDA、デザインパターン、ペトリネット、Machinations）
